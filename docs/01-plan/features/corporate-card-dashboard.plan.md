# 사내법인카드 관리 대시보드 (Corporate Card Dashboard) Plan Document

> Version: 1.0.0 | Created: 2026-03-06 | Status: Draft

## 1. Executive Summary
사내 직원들이 매월 카드사로부터 받는 PDF 사용내역 파일을 업로드하면, 지정된 엑셀 양식으로 자동 변환하고 관리자가 전체 사용 내역을 대시보드에서 통합 관리할 수 있는 시스템입니다.

## 2. Goals and Objectives
- **PDF-to-Excel 자동화**: 수작업으로 입력하던 카드 사용 내역을 PDF 파싱을 통해 엑셀로 자동 변환하여 업무 효율 증대.
- **데이터 통합 관리**: 직원별/부서별 카드 사용 현황을 관리자가 한눈에 파악할 수 있는 대시보드 구축.
- **일관된 포맷 유지**: 회사에서 정한 특정 엑셀 양식에 맞춘 데이터 출력 보장.

## 3. Scope
### In Scope
- **암호화된 PDF 처리**: 암호가 걸린 PDF 파일의 복호화 및 데이터 추출.
- **PDF 파일 업로드 (월 1회)**: 단일 카드사(동일 포맷) PDF 내역서 업로드 기능.
- **데이터 파싱 및 엑셀 변환**: PDF 내 날짜, 가맹점, 금액, 승인번호 등을 추출하여 정해진 엑셀 템플릿에 맞춰 저장.
- **관리자 대시보드**: 업로드된 전체 데이터의 요약(총합, 부서별 통계) 및 목록 조회.
- **사용자 관리**: 직원(업로드/본인 내역 조회) 및 관리자(전체 조회/엑셀 다운로드) 권한 분리.

### Out of Scope
- **실시간 카드 승인 연동**: 카드사 API 직접 연동은 제외(PDF 파일 기반).
- **다양한 카드사 대응**: 우선 현재 사용 중인 카드사 1개 모델만 집중 구현.
- **경비 정산 프로세스 연동**: 회사 내 ERP나 회계 시스템과의 직접 연동은 추후 검토.

## 4. Success Criteria
| Criterion | Metric | Target |
|-----------|--------|--------|
| 파싱 정확도 | 데이터 추출 성공률 | 99% 이상 (정해진 포맷 내) |
| 변환 속도 | PDF 업로드 후 엑셀 생성 시간 | 5초 이내 |
| 사용자 만족도 | 기존 수작업 대비 시간 단축율 | 80% 이상 절감 |

## 5. Timeline (Draft)
| Milestone | Date | Description |

| **Plan & Design** | Week 1 | 요구사항 정의 및 DB/API 설계 |

## 6. Risks
| Risk | Impact | Mitigation |
|------|--------|------------|
| PDF 암호 불일치 | 파싱 실패 (Medium) | 사용자에게 명확한 암호 오류 메시지 제공 및 재입력 유도 |
| PDF 레이아웃 변경 | 파싱 로직 오류 발생 (High) | 카드사의 레이아웃 변경 시 즉시 수정 가능한 모듈화된 파서 구조 설계 |
| 개인정보 노출 | 민감 정보 유출 (High) | 카드번호 마스킹 및 강력한 권한 관리 적용 |
| 엑셀 포맷 복잡성 | 출력 양식 불일치 (Medium) | 엑셀 템플릿 기반 쓰기 라이브러리(exceljs 등) 사용 |
